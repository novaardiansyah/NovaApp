# Pengenalan

- React Native - Expo
- REST API Laravel-12 (Sanctum)

# Instruksi

Kerjakan setiap taks yang tertera pada file `.docs/task.md`, setiap task memiliki scope/workspace yang membatasi jadi hanya modifikasi/kerjakan sesuai dengan batasan, dan jika ada dokumentasi/planning maka buat file markdown dan simpan ke direktori `.docs/*`.

Setiap poin yang terselesaikan pada `.docs/task.md`, maka beri tanda (DONE) di akhir poin, dan untuk respon pada terminal cukup beri tahu "task selesai"

WAJIB: kode program harus bersih dari komentar, better align (tab sejajar bila perlu), dan space/tab size=2.

# Batasan

Perintah seperti `artisan serve` atau `artisan migrate` tidak di izinkan, segala test akan dilakukan manual.
